# 🌸 Perceptron Model on Iris Dataset

An interactive and educational implementation of the **Perceptron Algorithm** from scratch using Python. This project demonstrates how a simple linear classifier learns to separate classes from the famous Iris dataset.

---

## 📌 Project Overview

This project focuses on:

* Implementing a **Perceptron classifier from scratch**
* Training on the **Iris dataset**
* Visualizing:

  * Training errors over epochs 📉
  * Decision boundaries 🎯
* Understanding how linear classifiers work in **machine learning fundamentals**

---

## 🧠 What is a Perceptron?

The Perceptron is one of the earliest supervised learning algorithms used for **binary classification**. It learns a decision boundary by updating weights based on misclassified samples.

---

## 📂 Dataset

* 🌼 **Iris Dataset** (from UCI Repository)
* Features used:

  * Sepal Length
  * Petal Length
* Target:

  * Flower species (converted into numerical classes)

## 🔍 Model Details

### 🔹 Parameters

| Parameter      | Description               |
| -------------- | ------------------------- |
| `eta`          | Learning rate             |
| `n_iter`       | Number of training epochs |
| `random_state` | Seed for reproducibility  |

---

### 🔹 Training Process

* Initialize weights randomly
* For each epoch:

  * Predict output
  * Update weights if misclassified
* Track number of errors
